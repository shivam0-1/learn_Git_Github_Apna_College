Git & GitHub Notes (Basics to Clone & Status)

1. What is Git?
Git is a Version Control System (VCS).
It tracks changes in files.
Helps manage project history.
Allows multiple developers to work together.
Benefits
Track changes
Restore previous versions
Team collaboration
Branching and merging

2. What is GitHub?
GitHub is a cloud platform for hosting Git repositories.
Used to store, share, and collaborate on projects online.

Git = Tool

GitHub = Online storage for Git repositories


3. Repository (Repo)
A repository is a folder where Git stores:
Project files
Version history
Commits

Repositories can be:
Local Repository (on your computer)
Remote Repository (on GitHub)

4. Local Repository
Stored on your computer.
You can work without internet.
Example:
MyProject/


5. Remote Repository
Stored on GitHub.
Used for backup and collaboration.
Example:
https://github.com/username/project-name

Git Workflow
Working Directory
        ↓
Git Repository (Local)
        ↓
GitHub Repository (Remote)
Install Git

Download from:
https://git-scm.com/

Check installation:
git --version

Example:
git version 2.45.0


Configure Git (First Time)
Set username = git config --global user.name "Shivam"

Set email= git config --global user.email "example@gmail.com"

Check configuration
git config --list


Clone Repository
What is Clone?
Clone means copying a GitHub repository to your local computer.

Command:
git clone <repository-link>

Example:
git clone https://github.com/shivam0-1/learn_Git_Github_Apna_College.git

What happens?
Downloads the project.
Downloads Git history.
Creates a local repository.
Connects it to the remote GitHub repository.


Git Status
Purpose
Shows the current state of the repository.

Command:  git status
It tells you:
Current branch
Modified files
New files
Deleted files
Files ready to commit (staged)
Files not yet staged (unstaged)

Example:
On branch main
nothing to commit, working tree clean

Example after editing:
Changes not staged for commit:
    modified: index.html
